# Company Site

会社サイト（Astro + Sanity）

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```

## デプロイ

- **サイト**: Vercel にデプロイ
- **Studio**: Sanity Studio にデプロイ済み

## 環境変数

プロジェクトルートに `.env.local` を作成（必要な場合）:

```bash
# Sanity
SANITY_STUDIO_PROJECT_ID=vdcrzio7
SANITY_STUDIO_DATASET=production
```
